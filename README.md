# NexoPark - Versión de Tesis

Aplicación móvil Android para la gestión de parqueos vehiculares, desarrollada en Kotlin con Jetpack Compose, Firebase y Google Maps.

Este repositorio contiene una versión limpia del código fuente para revisión académica. 
No incluye archivos sensibles como `google-services.json`, `local.properties`, certificados de firma ni claves privadas.
